# Blockchain Learning

Repositorio personal para aprender blockchain, Web3 y Rust paso a paso.

## Objetivo

Aprender de forma práctica y profesional los fundamentos de blockchain, el desarrollo Web3 y la transición hacia Rust y ecosistemas como Solana.

## Enfoque

Este proyecto sigue una metodología de aprendizaje paso a paso:

1. Entender los fundamentos
2. Construir ejemplos pequeños
3. Practicar con proyectos mínimos
4. Mejorar estructura, debugging y buenas prácticas
5. Escalar progresivamente a herramientas más avanzadas

## Ruta de aprendizaje inicial

1. JavaScript
2. Node.js
3. Solidity
4. ethers.js
5. Rust
6. Cargo
7. Solana CLI
8. Anchor
